メニューボタン（画面右のボタン）の説明

●時計
●絵記号追加： jpg画像の絵記号を追加できます。
　※追加ファイル名はひらがなにする必要があります。
●設定： 絵記号表示の設定が出来ます。
●閉じる： ソフトの終了


ソフトを使いやすいように設定する。

●設定1　
1. 選択モード：OFFにすると、色枠の移動が無くなりマウス操作モードになります。
2. 行数： 絵記号の並ぶ行数を3行と4行に変更できます。
3. 目次： 表示したい目次分野をチェックし、選択できます。
 　　　　 絵記号をクリックすると、目次とタブの画像を変更できます。

●設定2
1. 詳細表示： 説明・絵記号・絵記号一時で選択できます。
2. 移動速度： 色枠の移動速度を変更できます。



画像を追加する。

●絵記号詳細画面の絵記号（画像）を設定する。
1. 絵記号詳細画面を開き「no img」をクリックする。
2. 保存ダイアログが表示される。
3. パソコン内の画像ファイルを選択する。（画像はjag画像のみです）
4. 絵記号が張り付く。


●絵記号を追加する。
1. ツール（画面右の縦に並んだボタン）の「+車椅子マーク」をクリックし、絵記号追加画面を表示する。
2. 追加したい分野の絵記号（中央の青い領域）をクリックする。
3. 保存ダイアログが表示される。
4. パソコン内の画像ファイルを選択する。（画像はjag画像のみです）
5. 絵記号会話ソフトを一旦閉じ、再度起動すると絵記号画像が反映されます。
※目次画像は、他の画面で変更できます。画像はここで追加しておく必要があります。
 画像ファイル名が目次画面の項目名として表示されます。


●目次画像を変更する。
1. ツール（画面右の縦に並んだボタン）の「+歯車マーク」をクリックし、設定画面を表示する。
2. 設定画面の設定1タブの上から3番目「目次：」で変更したい目次画像をクリックする。
3. フォルダー内（tab1）が表示されるので、変更したい画像選択し、開くボタンを押す。
4. 画面を閉じ、絵記号ソフトを一旦閉じ、再度起動すると設定が反映されます。
※また、「目次：」のチェックマークをクリックするとチェックが外れ、その項目のタブは次回起動からは表示されなくなります。
 （チェックを入れなおすとタブは再び表示されます。）
