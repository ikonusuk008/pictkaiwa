絵記号会話ソフト MIKAN Software
脳性麻痺、筋委縮性側索硬化症（ALS）の方のための絵(絵記号)による会話補助ソフトです。
絵記号での会話では、紙のボードに書かれた絵を指差すことによって会話されています。
この作業をパソコン画面を通じて障害者の会話を補助します。 パソコン画面なので今までの紙のボードを介護者が支える必要がなく、ご自身で操作出来ます。 ボタン操作が簡単で、粗雑な押し方でも操作来ます。 ボタン操作はテンキーを使用し、テンキーのどこを押しても同じ操作が出来ます。 基本的な日常会話が出来るよう工夫しております。

絵記号作成
絵記号画像は購入時、基本的なものが付属されていますが、
独自の絵記号を使いたい場合は、1画像300円で作成いたします。
目次画面 （以下基本的な使い方）
1.　身体・体調・動作・食事等にカテゴリー分けされた目次が絵の表で表示されます。

2.　選択枠が順々に自動で移動します。

3.　目的の絵のところに選択枠がきたらテンキーを押してください。

4.　テンキーにあるどのキーを押しても構いません。

5.　選択枠が止まり、目的のカテゴリタブ（分野別に分かれた画面）に移動します。

※テンキーのNumLockボタンはボタンとして働きませんので、NumLockが外側に付いているテンキーを使用してください。
NumLockが外側に付いているテンキー（Amazon）

下の図では、カテゴリ「動作」に選択枠（ピンク色の枠）がきています。
選択枠は、カテゴリを順番に移動しています。
この時にボタン（テンキーのどのボタンでもよい）を押すと動作のカテゴリーが選択され絵記号画面へ移ります。
絵記号画面
次に絵記号画面が表示されます。

＜列の選択＞
1.　絵記号画面（絵記号の分野別画面）では、行ごとに黄色い選択枠が移動します。

2.　目的の絵記号が含まれる列に選択枠が移動してきた時に、テンキー（テンキーのどのボタンでもよい）を押して下さい。

＜絵記号(絵)の選択＞
1.　行を選ぶと、次は赤い選択枠が左→右に移動していきます。

2.　目的の絵記号の所に赤い選択枠がきたらテンキー（テンキーのどのボタンでもよい）を押してください。
　　選択枠が止まり、目的の絵記号が選択されます。

下の図では、２行目がすでに選択され、赤い選択枠が左→右に移動しているところです。
ぱんの所に選択枠がきています。

絵記号(絵)の説明画面
1.　次に説明画面（絵記号の意味を具体化する画面）が表示されます。

2.　再生ボタンを押すと、表示されている文章や単語が音声で読み上げられます。
　　（この機能は、ソフト利用者と介助者が長時間の正確な意思疎通するのに役立つ場合があります）

3.　説明の列には、テキストを入力・編集・保存・削除ができます。
　　（テキストは説明画面を閉じたときに保存されます。）

4.　2回テンキー（テンキーのどのボタンでもよい）を押すと説明画面は閉じられ、再び絵記号画面が表示されます。

　　下の図では、「ぱん」の「説明画面」が表示されています。
　　再生ボタンを押すと文を読み上げます。

メニューボタン（画面右のボタン）の説明
1.　時計
　　・大きく年月日・曜日を表示されます。
　　・大きくデジタルとアナログの時計表示されます。

2.　絵記号追加
　　 jpg（ファイルの種類）の絵記号を追加できます。
　　※追加ファイル名はひらがなにする必要があります。

3.　設定
　　各設定が行えます。

4.　閉じる
　　ソフトを終了します。
ソフトを使いやすいように設定する。
●設定１
1.　選択モード 　　OFFにすると、色枠の移動が無くなりマウス操作モードになります。
　　絵記号の移動色枠をタイミングにより選択できない場合に使います。 　　

2.　行数
　　絵記号カテゴリ画面の行数を3行と4行に変更できます。

3.　目次
　　表示したい目次（絵記号カテゴリ）をチェックし、選択できます。
　　絵記号をクリックすると、目次画像（絵記号）を変更できます。



●設定２
1.　詳細表示
　　説明：　絵記号について詳細データを表示します。
　　絵記号：　絵記号のみ大きく表示されます。
　　絵記号一時：　絵記号のみ大きく表示され、５秒間表示後、自動で消えます。

2.　移動速度： 絵記号の移動色枠の速度を変更できます。


画像を追加する
■カテゴリ別に絵記号を追加します。
画像（絵記号）はjpgである必要があります。
jpgとは、最も多く使われる画像形式で、ファイル名の後に.jpgとついているものです。
ペイントソフトで作ったり、インターネットで無料ダウンロードできます。
これを、パソコンの任意のフォルダーに保存してください。
そして、以下の方法で指定する分野に画像（絵記号）を入れることが出来ます。

1.　「+車椅子マーク」をクリックし、絵記号追加画面を表示する。

2.　追加したい分野の絵記号（中央の青い領域）をクリックする。
　　（下の「＿詳細」が付いたボタンは、絵記号詳細画面の追加画像を保存できます） 　

3.　保存ダイアログが表示される。
　　（※画像はjag画像のみ表示出来ます。）
　　あらかじめ保存しておいた任意の画像ファイル内の画像を選択してください。
　　「絵記号ソフト自身の画像」（以下で説明）を選択しても画像は追加させません。
　　（直接フォルダに画像（絵記号）を入れる事によっても、追加できます）
　　ファイルパス：　ソフト本体の　「pict_kaiwa_soft\resource\img」内の以下に有ります。
　　tab1 (目次)
　　tab2 (体調)
　　tab2_詳細 (体調の詳細)
　　tab3 (身体)
　　tab3_詳細 (身体の詳細)
　　tab4 (人)
　　tab4_詳細 (人の詳細)
　　tab5 (動作)
　　tab5_詳細 (動作の詳細)
　　tab6 (物)
　　tab6_詳細 (物の詳細)
　　tab7 (食事)
　　tab7_詳細 (食事の詳細)
　　tab8 (時間)
　　tab8_詳細 (時間の詳細)
　　tab9 (場所)
　　tab9_詳細 (場所の詳細)
　　tab10 (アプリ)
　　tab10_詳細 (アプリの詳細)

4.　パソコン内の画像ファイルを選択する。（画像はjag画像のみです）

5.　絵記号会話ソフトを一旦閉じ、再度起動すると絵記号画像が反映されます。

以下は、カテゴリごとの ファイル選択・保存ダイアログが表示されています。


■絵記号の説明画面の画像を追加します。
1.　左のno img の部分をクリックするとファイル選択が出ます。

2.　絵記号の追加と同じ要領でパソコン内の画像ファイルを選択してください。 　　no img の部分に画像が挿入されます。

以下は、絵記号説明画像の ファイル選択・保存ダイアログが表示されています。


■目次画像を変更します。
1.　ツール（画面右の縦に並んだボタン）の「+歯車マーク」をクリックし、設定画面を表示する。

2.　設定画面の [設定1] タブの上から3番目「目次：」で変更したい目次画像上をクリックする。

3.　フォルダー内（tab1）が表示されるので、変更したい画像選択し、[開く]ボタンを押す。

4.　設定画面と絵記号会話ソフトを一旦終了し、再度 絵記号会話ソフト起動すると設定が反映されます。
　　（※また、「目次：」のチェックマークをクリックするとチェックが外れ、その項目のタブは次回起動からは表示されなくなります。

以下は、目次画像の ファイル選択・保存ダイアログが表示されています。
